# Todo-List-Vanilla-JS

Test task for Salt &amp; Pepper.

The goal is to create common todo-list with filter and counter using only pure javascript. Any libraries or frameworks are prohibited.

[Click here to try it! Deployed on Vercel.](https://todo-list-vanilla-js-test-task.vercel.app/)

## Install

```sh
$ make install
```

## Start app 
####  *(open http://localhost:4242)*
```
$ make serve
```
